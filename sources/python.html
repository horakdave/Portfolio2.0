Projekt-TNP

Projekt-TNP je software pro pomoc s vyděláváním peněz pomocí akciového/kryptoměnového daytradingu. Pomůže vám najít správnou chvíli, kdy koupit/porodat, aby jste vydělali co nejvíce peněz. Poslední verze TNP-graphing0.31 používá taktiku "Simple Moving Average" strategii, kde je kromě ceny vyzobrazen průměr krátkého časového období(Short MA) a průměr dlouhého časového období(Long MA). Pokud se průměry setkají, tak software nakoupí/prodá. Projekt TNP je zatím můj největší projekt a pracuji na něm už od října 2023. Pro více detailů o projektu se můžete podívat na můj Github
